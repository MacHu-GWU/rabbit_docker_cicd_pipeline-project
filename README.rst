Install
------------------------------------------------------------------------------

``rabbit_docker_cicd_pipeline`` is released on PyPI, so all you need is:

.. code-block:: console

    $ pip install rabbit_docker_cicd_pipeline

To upgrade to latest version:

.. code-block:: console

    $ pip install --upgrade rabbit_docker_cicd_pipeline
